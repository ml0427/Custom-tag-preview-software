# 專案啟動說明 (Startup Guide)

這份文件提供給開發人員或一般使用者，用來啟動「自定義標籤預覽軟體」的前後端服務。

## 🌟 最快的方式：一鍵啟動 (One-Click Start)
在專案資料夾底下，已經為您準備好了 `start.bat` 檔案。
**只要滑鼠對著 `start.bat` 連點兩下 (雙擊)**，它就會自動幫您同時開啟後端與前端的伺服器！

---

## 另外的方法：手動分別啟動
如果您想要自己手動打指令，請接著往下看：

### 1. 啟動後端伺服器 (Spring Boot)

後端伺服器負責掃描 `server/comic_storage/` 資料夾內的 ZIP 檔，並提供 API 服務（預設執行於 `http://localhost:8080`）。

### Windows 啟動步驟：
1. 打開 PowerShell 或命令提示字元 (Terminal)。
2. 切換到 `server` 資料夾：
   ```powershell
   cd server
   ```
3. 設定 Java 環境變數並啟動 (如果您已經有全域的 JAVA_HOME 則不需要第一行)：
   ```powershell
   $env:JAVA_HOME="C:\Users\ml042\.jdks\temurin-17.0.13"
   .\mvnw spring-boot:run
   ```
4. 看到 `Started ComicManagerApplication` 即代表啟動成功。您可以在 `server/comic_storage/` 中放入新的 ZIP 漫畫檔，系統每 60 秒會自動掃描進資料庫。

---

## 2. 啟動前端介面 (Vue 3 + Vite)

前端介面提供視覺化的畫廊、標籤管理與詳情預覽（預設執行於 `http://localhost:5173`）。

### Windows 啟動步驟：
1. 打開**另一個新的** PowerShell 或命令提示字元 (Terminal)。
2. 切換到 `client` 資料夾（若已建立）：
   ```powershell
   cd client
   ```
3. 安裝依賴套件 (初次啟動或更新套件時需要)：
   ```powershell
   npm install
   ```
4. 啟動開發伺服器：
   ```powershell
   npm run dev
   ```
5. 開啟瀏覽器並前往終端機顯示的網址（通常為 `http://localhost:5173`）即可開始使用軟體！

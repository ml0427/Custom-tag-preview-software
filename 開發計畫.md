# 自定義標籤預覽軟體 - 開發計畫

本文件記錄了目前待處理的 9 項優化需求與實作方向。

## 📋 待辦需求清單

1. **自動標籤邏輯自定義化**
   - 取代 `scanner.rs` 硬編碼邏輯，改為動態讀取資料庫 `tag_rules`。

2. **刪除功能擴充 (右鍵選單)**
   - 在左側 `WorkspacePanel.vue` 樹狀圖增加右鍵刪除選項。
   - 在右側 `ComicGallery.vue` 列表項目增加右鍵刪除選項。
   - 統一連動後端刪除邏輯。

3. **標籤篩選選中高亮修正**
   - 修正選中標籤時變暗的問題，提升對比度與視覺回饋。

4. **統一且明顯的刪除圖示**
   - 全軟體統一使用明顯的「X」圖示進行刪除/移除操作。

5. **掃描精靈支援子目錄選擇**
   - 讓掃描規則時的路徑選取器可以深入選擇子資料夾。

6. **標籤規則下拉選單樣式修正**
   - 修正設定介面中下拉選單 (Select) 的白色背景，使其符合深色主題。

7. **工作目錄功能工具列**
   - 新增：上一頁、上一層、重新整理、詳細訊息、清除搜尋等按鈕。

8. **搜尋列與工具列合併**
   - 整合搜尋輸入框與工具列按鈕的 UI 配置。

9. **樹狀圖選取連動右側預覽**
   - 在樹狀圖點選資料夾時，右側 Gallery 自動顯示預覽內容。
